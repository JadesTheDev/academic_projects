# Final Calculator Project

## Description
This is a C# console calculator application that performs basic operations and includes extended functionality such as storing and managing values using collections.

## Features
- Add, subtract, multiply, divide
- Store and retrieve values
- Modify and remove values
- Sum and average calculations
- Error handling for invalid input and divide-by-zero

## How to Run
1. Open in Visual Studio
2. Build and run the project
3. Use the menu to perform operations
